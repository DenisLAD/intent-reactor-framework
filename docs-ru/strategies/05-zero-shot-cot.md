# Стратегия Zero-Shot CoT

**Модуль:** `intent-reactor-strategies`
**Значение стратегии:** `zero-shot-cot`

---

## Краткое описание

Zero-Shot CoT добавляет к сообщению пользователя единственную триггерную фразу — *«Давай рассуждать пошагово»* — вызывая пошаговое рассуждение без каких-либо few-shot примеров в промпте. Это самая простая техника улучшения рассуждений в библиотеке.

---

## Принцип работы

1. Планировщик добавляет настроенную триггерную фразу к входному сообщению пользователя.
2. Выполняется один вызов LLM.
3. LLM выдаёт рассуждения, за которыми следует финальный ответ.
4. Выдаётся шаг `DONE`, ответ возвращается.

Как и [CoT](04-cot.md), Zero-Shot CoT — декоратор над ReACT и поддерживает инструменты.

---

## Конфигурация

```yaml
intent-reactor:
  planning:
    strategy: zero-shot-cot
```

Специфических свойств нет.

---

## Промпты

| Промпт | Путь в classpath |
|---|---|
| Триггерная фраза | `prompts/zero-shot-cot-trigger.md` |

Содержимое по умолчанию: `«Let's think step by step.»`

---

## Когда использовать

- Быстрые аналитические вопросы, требующие структурированного рассуждения, но без few-shot примеров.
- Прототипирование: самый дешёвый способ получить пошаговый вывод.
- Когда нужен CoT-стиль рассуждений без накладных расходов на составление примеров.

---

## Отличие от CoT

| | `cot` | `zero-shot-cot` |
|---|---|---|
| Добавление в промпт | Многострочный блок инструкций | Одна триггерная фраза |
| Few-shot примеры | Опционально (настраиваемо) | Никогда |
| Объём вывода | Обычно больше | Обычно лаконичнее |

---

## Пример

```yaml
intent-reactor:
  planning:
    strategy: zero-shot-cot
```

```java
String result = intentReactorService.process("session-1",
    "Сколько составляет 15% от 240?");
// Добавляется "Let's think step by step."
// LLM: "15% от 240 = 0.15 × 240 = 36."
// → возвращает "36"
```
