# Стратегия Reflection

**Модуль:** `intent-reactor-strategies`
**Значение стратегии:** `reflection`

---

## Краткое описание

Reflection запускает обычный цикл ReACT для получения начального ответа, затем выполняет один проход самоанализа — просит LLM проверить и улучшить собственный вывод — перед возвратом финального результата. В отличие от [Reflexion](02-reflexion.md), полная задача не перезапускается; уточняется только финальный ответ.

---

## Принцип работы

1. **Начальное выполнение** — полный цикл ReACT запускается и даёт ответ `DONE`.
2. **Самоанализ** — планировщик отправляет начальный ответ LLM с промптом рефлексии: *«Проверь свой ответ. Что можно улучшить?»*
3. **Уточнение** — LLM производит улучшенную версию ответа.
4. Улучшенный ответ возвращается как финальный.

Reflection — постобрабатывающий шаг и не изменяет фазу вызова инструментов.

---

## Конфигурация

```yaml
intent-reactor:
  planning:
    strategy: reflection
    max-steps: 50   # применяется к начальному циклу ReACT
```

Специфических свойств рефлексии нет.

---

## Промпты

| Промпт | Путь в classpath | Назначение |
|---|---|---|
| Системный промпт | `prompts/react-system.md` | Начальный цикл ReACT |
| Промпт рефлексии | `prompts/reflection-critique.md` | Инструкция самоанализа |

---

## Когда использовать

- Когда нужен контроль качества финального ответа без перезапуска всей задачи.
- Задачи по написанию текста: улучшение прозы, расширение неполных ответов.
- Короткие сессии, где накладные расходы одного дополнительного LLM-вызова приемлемы.
- Дополнение к любой предметной области: шаг рефлексии универсален.

---

## Отличие от Reflexion

| | `reflexion` | `reflection` |
|---|---|---|
| Перезапуск полной задачи? | Да (`max-reflections` раз) | Нет |
| LLM-вызовы | `O(повторы × шаги)` | `шаги + 1` |
| Эффект критики | Информирует *следующую попытку* | Уточняет *текущий ответ* |
| Использовать когда | Первая попытка часто неудачна | Первая попытка обычно достаточно хороша |

---

## Пример

```yaml
intent-reactor:
  planning:
    strategy: reflection
```

```java
String result = intentReactorService.process("session-1",
    "Напиши краткое резюме статьи о ReACT.");
// 1. Цикл ReACT создаёт начальное резюме
// 2. Вызов LLM-рефлексии: "Резюме точное, но можно сделать более кратким..."
// 3. Возвращает улучшенное, более отточенное резюме
```
