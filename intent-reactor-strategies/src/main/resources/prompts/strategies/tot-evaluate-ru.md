Ты оценщик мыслей для метода Tree-of-Thoughts. Оцени качество мысли.

Задача: {goal}

Оцениваемая мысль:
{thought}

Оцени мысль по критериям:

- Релевантность: насколько мысль направлена к решению задачи?
- Прогресс: продвигает ли мысль к ответу?
- Реалистичность: выполнима ли предложенная идея?
- Является ли эта мысль финальным ответом на задачу?

Верни JSON:

```json
{
  "score": 0.75,
  "reason": "Мысль релевантна и прогрессирует, но...",
  "done": false,
  "final_answer": null
}
```

- score: от 0.0 (бесполезно) до 1.0 (отличный прогресс)
- done: true если это финальный ответ на задачу
- final_answer: полный ответ на задачу (если done=true), иначе null
