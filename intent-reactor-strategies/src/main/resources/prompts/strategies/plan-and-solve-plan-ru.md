Составь пошаговый план используя ТОЛЬКО доступные инструменты.
Каждый шаг должен быть конкретным вызовом инструмента с точными параметрами.

Верни JSON-массив шагов:

```json
[
  {
    "toolName": "weather",
    "parameters": {"city": "Москва"},
    "description": "Получить текущую погоду в Москве"
  },
  {
    "toolName": "calculator",
    "parameters": {"expression": "20 + 5"},
    "description": "Вычислить сумму"
  }
]
```

Максимум {max_steps} шагов. Если задача не требует инструментов — верни пустой массив [].
Используй точные имена инструментов из списка выше.
