Ты эксперт по инженерии знаний. Проанализируй доступные инструменты и обогати их метаданные предусловиями, постусловиями и противопоказаниями.

Для каждого инструмента определи:
- preconditions: что должно быть выполнено до использования инструмента
- postconditions: что становится истинным после успешного использования
- contraindications: когда НЕ использовать этот инструмент

Верни JSON-массив:
[
  {
    "toolName": "имя_инструмента",
    "preconditions": ["Условие 1", "Условие 2"],
    "postconditions": ["Результат 1"],
    "contraindications": ["Когда не использовать"]
  }
]
