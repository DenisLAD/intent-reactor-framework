# Стратегия Chain-of-Thought (CoT)

**Модуль:** `intent-reactor-strategies`
**Значение стратегии:** `cot`

---

## Краткое описание

Цепное рассуждение (Chain-of-Thought) инструктирует LLM рассуждать пошагово перед выдачей окончательного ответа. В отличие от ReACT, CoT не использует цикл вызова инструментов — выполняется один LLM-вызов с промптом, ориентированным на рассуждение, и ответ возвращается сразу.

---

## Принцип работы

1. Планировщик CoT добавляет инструкцию цепного рассуждения к промпту: *«Рассуждай пошагово перед ответом»*.
2. Выполняется один вызов LLM. Вывод модели содержит как промежуточные рассуждения, так и финальный ответ.
3. Планировщик выдаёт шаг `DONE` с полным ответом как выводом.
4. Ответ возвращается немедленно; цикл `max-steps` не запускается.

Поскольку CoT — декоратор над ReACT, можно **совмещать** CoT с инструментами: планировщик CoT генерирует начальный план рассуждений, затем передаёт управление циклу ReACT для вызова инструментов в соответствии с этим планом.

---

## Конфигурация

```yaml
intent-reactor:
  planning:
    strategy: cot
```

Специфических свойств нет. `max-steps` не влияет на чистый CoT (нет цикла итераций).

## Когда использовать

- Математические задачи, логические головоломки, аналитические вопросы, где демонстрация хода решения повышает точность.
- Вопросы, где LLM обладает достаточными знаниями без внешних инструментов.
- Когда важна латентность и решение в один вызов приемлемо.
- Как строительный блок: включите инструменты вместе с CoT для гибридных пайплайнов «рассуждай, затем действуй».

---

## Сравнение плюсов и минусов

| Преимущества | Недостатки |
|---|---|
| Один вызов LLM — быстро и дёшево | Нет доступа к инструментам по умолчанию |
| Выше точность, чем zero-shot, на сложных рассуждениях | Зависит исключительно от параметрических знаний LLM |
| Просто включить | Может давать многословный вывод |

---

## Пример

```yaml
intent-reactor:
  planning:
    strategy: cot
```

```java
String result = intentReactorService.process("session-1",
    "Если поезд проезжает 120 км за 1.5 часа, какова его средняя скорость?");
// LLM рассуждает: "Скорость = расстояние / время = 120 / 1.5 = 80 км/ч"
// → возвращает "Средняя скорость составляет 80 км/ч."
```
