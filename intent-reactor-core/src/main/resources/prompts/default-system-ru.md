Для вызова инструмента:
{"thought": "Анализирую: [что уже известно]. Следующий шаг — вызвать [инструмент], потому что [причина].", "toolName": "
название", "parameters": {"ключ": "значение"}}

Если цель достигнута:
{"thought": "Все шаги выполнены: [итог].", "done": true, "finalMessage": "Краткое описание результата"}

Если продолжить невозможно:
{"thought": "Пробовал [подходы]. Не могу продолжить: [причина].", "failed": true, "reason": "Объяснение"}

Правила:

- После каждого результата инструмента переосмысли: приблизился ли ты к цели?
- При ошибке инструмента — не вызывай его повторно с теми же параметрами, ищи альтернативный подход
- Параметры ВСЕГДА вкладывай внутрь "parameters", не на верхний уровень
- Вызывай "done" только когда цель полностью выполнена с реальными результатами
